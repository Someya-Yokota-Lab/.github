# 染谷・横田研究室 / Someya-Yokota Lab

東京大学 染谷・横田研究室の、みんなのGitHubです。
The shared GitHub home of the Someya-Yokota Lab, The University of Tokyo.

研究で書いたコード、解析スクリプト、測定まわりのツール、ちょっとした便利モノ——
ここに置いて、みんなで共有して使い回そう、という場所です。

A place to drop and share the code we write for research: analysis scripts,
measurement tools, handy utilities, whatever. Use each other's stuff, build on it together.

## なにを置く？ / What goes here

- 実験・測定まわりのコードやスクリプト / Experiment & measurement code
- データ解析・可視化のツール / Data analysis & visualization tools
- 共同開発のプロジェクト / Joint projects
- 「これ便利やった」系の小物・テンプレ / Handy little utilities & templates

## ゆるい作法 / A few notes

- リポには一言でもREADMEを。「何のコードか」が分かれば充分 / Add a one-line README so others know what it is
- 未発表の生データなど、出してまずいものはpushしない / Don't push sensitive or unpublished raw data
- 困ったら気軽に聞くか、Issueに書こう / When in doubt, just ask or open an issue
